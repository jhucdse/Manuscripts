This repository includes codes for papers that used data from the Johns Hopkins Precision Medicine Analytics Platform (PMAP).

The folder titled "2025_PDS_Heyward" contains R codes written by James Heyward (PhD, 2025) for Validation of Immune-Related Adverse Event (irAE) Case Definitions in a Real-World Lung Cancer Population. 
The analysis was published in Pharmacoepidemiology and Drug Safety in 2025 (PubMed link: https://pubmed.ncbi.nlm.nih.gov/34806036/).

Full Citation: Heyward JS, Segal JB, Mehta HB, Murray JC. Validation of Immune-Related Adverse Event (irAE) Case Definitions in a Real-World Lung Cancer Population. Pharmacoepidemiol Drug Saf. 2025 Feb;34(2):e70100. doi: 10.1002/pds.70100. PMID: 39961795; PMCID: PMC11970256.
